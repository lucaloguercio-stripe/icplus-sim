# IC+ Fee Simulator - Stripe App

Stripe App that generates activity reports with simulated IC+ (Interchange Plus) fee breakdowns.

## Prerequisites

1. Install the [Stripe CLI](https://stripe.com/docs/stripe-cli)
2. Login: `stripe login`

## Install

```bash
npm install
```

This installs both the Stripe App and backend dependencies (backend is installed automatically via `postinstall`).

## Run

Start backend + frontend together:

```bash
npm start
```
